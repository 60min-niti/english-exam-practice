## ▶ แผนเช้าพรุ่งนี้ (ตามที่ผู้ใช้เลือก)
1. ✅ **โหมดฝึกแยกตามหัวข้อ (tag)** — เสร็จแล้ว: หน้า "ฝึกแยกหัวข้อ" (nav + การ์ดหน้าหลัก) เลือกหัวข้อจัดกลุ่มตามพาร์ท ดึงข้อข้ามทุกชุดมาฝึกแบบสุ่ม เฉลยทันที + แนะนำหัวข้อจุดอ่อนจากสถิติ (ไม่นับรวมในสถิติหลัก/ประวัติ)
2. ✅ **QA ตรวจความถูกต้อง 500 ข้อ (second-pass)** — เสร็จแล้ว: ตรวจเชิงระบบ (โครงสร้าง EI/ขีดเส้นใต้/ช่องว่าง/ตัวเลือกซ้ำ/คำตอบ↔เฉลย/คำแปลซ้ำ) = 0 ข้อผิดจริง
   - ✅ ใส่ **หมายเหตุ** กำกับข้อสำเนียงภาษา (ST6-09 `data suggest`, ST7-11 `committee are`) แล้ว — เก็บข้อไว้ ระบุว่าเป็นรูปวิชาการ/บริติช
- จุดบกพร่องอื่น (เกมตัวลวงซ้ำ, ดูเฉลยย้อนหลัง, streak, history cap) → **ไว้ค่อยแก้ทีหลัง**
- ดีไซน์หน้าเว็บ (Figma/ธีม/มือถือ/ฟอนต์) → ยังไม่แตะ

## 🐞 จุดบกพร่อง / ความเสี่ยง

- [ ] **เกมทายคำแปล: ตัวลวงอาจซ้ำความหมายคำตอบ** — สุ่มตัวลวงแบบสุ่มล้วน คลังมีคำที่แปลเหมือนกันหลายคำ (alleviate/ease/relieve = บรรเทา) → กันตัวลวงที่ความหมายใกล้คำตอบ (เทียบคีย์เวิร์ดไทย/ดึงจาก tag ต่างกัน)
- [ ] **ดูเฉลยย้อนหลังไม่ได้** — history เก็บแค่ score/time/perTag ไม่เก็บคำตอบรายข้อ → เพิ่มการเก็บ answers ต่อ attempt + หน้าเปิดดูเฉลยรอบเก่า
- [ ] **ข้อขึ้นกับสำเนียง** — ST6-09 (data suggest), ST7-11 (committee are) ถูกเชิงวิชาการ/บริติช แต่ตำราไทยบางเล่มสอนต่าง → ตัดสินใจเก็บเป็น "ระดับสูง" (ใส่หมายเหตุ) หรือปรับ
- [ ] **streak สมุดข้อผิดทำงานเฉพาะโหมด retry** — ตอบถูกในโหมดฝึกปกติไม่ปลดข้อ → ให้ตอบถูกที่ไหนก็นับ streak
- [ ] **ยังไม่ได้ QA ครบ 500 ข้อ** — ไล่แก้ข้อ "ทุกตัวเลือกถูก" ไปแล้ว 8 ข้อ (ST4-24, ST5-23, ST6-24, ST7-19/21/23, ST8-15/23) แต่ต้องตรวจ second-pass: คำตอบถูกต้อง, EI มีจุดผิดเดียว, คำแปลตรงบริบท
- [ ] **history ไม่จำกัดขนาด** — ใส่เพดาน/เก็บแบบสรุป
