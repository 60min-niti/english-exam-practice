# TU-GET PBT Practice

เว็บฝึกทำข้อสอบแนว TU-GET PBT แบบ Static — เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร ไม่ต้องต่ออินเทอร์เน็ต

> **Disclaimer:** เว็บนี้เป็นแบบฝึกหัด "แนวข้อสอบ" ที่แต่งขึ้นใหม่ทั้งหมดเพื่อการศึกษา ไม่ใช่ข้อสอบจริง และไม่มีส่วนเกี่ยวข้องกับมหาวิทยาลัยธรรมศาสตร์หรือสถาบันภาษา มธ.

## โครงสร้างข้อสอบ

| พาร์ท | จำนวนข้อ | แบ่งย่อย | เวลา (โหมดสอบ) | คะแนน |
|---|---|---|---|---|
| Structure | 25 | Sentence Completion 13 + Error Identification 12 | 25 นาที | 250 |
| Vocabulary | 25 | Cloze 12 + Synonym 13 | 25 นาที | 250 |
| Reading | 50 | 5 passages × 10 ข้อ | 65 นาที | 500 |
| **เต็มชุด** | **100** | | **3 ชั่วโมง** | **1,000** |

ข้อละ 10 คะแนน ทุกข้อมีเฉลยพร้อมคำอธิบายภาษาไทย

## ฟีเจอร์

- ทำเต็มชุด 100 ข้อ หรือเลือกทำรายพาร์ท เลือกชุดข้อสอบได้
- **โหมดสอบ** (จับเวลา นับถอยหลัง หมดเวลาส่งอัตโนมัติ) / **โหมดฝึก** (ไม่จับเวลา เฉลยทันทีหลังตอบ)
- บันทึกอัตโนมัติ — ปิดเบราว์เซอร์กลางคันแล้วกลับมาทำต่อได้
- สถิติ + กราฟพัฒนาการ + จุดอ่อนรายหัวข้อ (แท็กไวยากรณ์/ประเภทคำถาม)
- สมุดข้อผิด: ข้อที่ตอบผิดเข้าคลังอัตโนมัติ ทำซ้ำจนตอบถูก 2 ครั้งติดถึงหลุดจากคลัง
- Export / Import ข้อมูล (ย้ายเครื่อง/สำรองข้อมูล) — ข้อมูลทั้งหมดเก็บใน LocalStorage ของเบราว์เซอร์
- โหมดมืด

## วิธีเพิ่มชุดข้อสอบใหม่

1. คัดลอกไฟล์ในโฟลเดอร์ `data/` เช่น `structure-1.js` → `structure-4.js`
2. แก้ `set: 1` เป็น `set: 4` แล้วเปลี่ยนโจทย์ทั้งหมด (รักษาโครงสร้าง field เดิม: `id`, `type`, `tag`, `question`, `choices`, `answer` (index 0–3), `explanation`)
3. เพิ่ม `<script src="data/structure-4.js"></script>` ใน `index.html` (ก่อน `js/app.js`)
4. รีเฟรชหน้าเว็บ — ชุดใหม่จะโผล่ในตัวเลือกอัตโนมัติ

รูปแบบไฟล์ Reading ต่างจากพาร์ทอื่นเล็กน้อย: เป็น `passages: [...]` โดยแต่ละ passage มี `title`, `text` และ `questions` ของตัวเอง
